Quickstart

Install

dependencies:
  smart_refresher: ^0.2.0
flutter pub get

Minimal setup

final controller = RefreshController();

SmartRefresher(
  controller: controller,
  onRefresh: () async {
    await Future.delayed(const Duration(seconds: 1));
    controller.refreshCompleted();
  },
  child: ListView.builder(itemBuilder: (_, i) => ListTile(title: Text('Item $i'))),
)
`controller` tracks state; call `refreshCompleted`/`loadComplete` to end animations.

Patterns

  • Refresh only: `enablePullDown: true`, `enablePullUp: false`.
  • Load more only: `enablePullDown: false`, `enablePullUp: true`.
  • Both: enable both callbacks.
