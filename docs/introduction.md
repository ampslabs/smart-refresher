`smart_refresher` is a powerful and highly customizable Flutter package that provides pull-to-refresh and infinite scrolling functionality for any scrollable widget. It is an actively maintained fork of the popular `flutter_pulltorefresh` library, rebranded and updated for modern Flutter development.

## Core Features

<CardGroup cols={2}>
  <Card title="Multi-Style Indicators" icon="palette">

  </Card>
  <Card title="Flexible Containers" icon="boxes-stacked">

  </Card>
  <Card title="Global Configuration" icon="gear">
    Configure default behavior once and apply it across your entire application.

## Why smart_refresher?

This fork was created to provide a stable, well-documented, and actively maintained version of the original pull-to-refresh library. We focus on:

- **Flutter Compatibility**: Ensuring full support for the latest Flutter stable releases (3.10+).
- **Modern Standards**: Using modern Dart features like super parameters and strict null safety.
- **English Documentation**: Providing comprehensive documentation and comments in English.
- **Community Driven**: Actively responding to issues and pull requests.
