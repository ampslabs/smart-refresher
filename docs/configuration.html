Configuration

RefreshConfiguration

PropertyTypeDefaultDescription
headerBuilderIndicatorBuilder?nullGlobal default header.
footerBuilderIndicatorBuilder?nullGlobal default footer.
dragSpeedRatiodouble1.0Overscroll drag multiplier.
headerTriggerDistancedouble80.0Refresh trigger threshold.
footerTriggerDistancedouble15.0Load trigger threshold.
twiceTriggerDistancedouble150.0Two-level open threshold.
closeTwoLevelDistancedouble80.0Two-level close threshold.
enableScrollWhenTwoLevelbooltrueAllow scroll in second floor.
enableScrollWhenRefreshCompletedboolfalseAllow dragging during completed stage.
enableBallisticRefreshboolfalseTrigger during ballistic scroll.
enableBallisticLoadbooltrueLoad during ballistic scroll.
enableLoadingWhenFailedbooltruePermit retry by pull up.
enableLoadingWhenNoDataboolfalsePermit loading in noMore state.

SmartRefresher props

Core props: controller, child, header, footer, enablePullDown, enablePullUp, enableTwoLevel, onRefresh, onLoading, onTwoLevel, plus scroll view passthrough fields like physics, reverse, primary, cacheExtent, semanticChildCount, center.

RefreshController methods

MethodDescription
requestRefresh()Programmatically starts header refresh.
requestLoading()Programmatically starts footer load.
requestTwoLevel()Opens second-floor panel.
refreshCompleted/refreshFailed/refreshToIdleHeader completion APIs.
loadComplete/loadFailed/loadNoData/resetNoDataFooter completion APIs.
