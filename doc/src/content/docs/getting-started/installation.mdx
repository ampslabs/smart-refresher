`smart_refresher` is a powerful Flutter widget designed to provide seamless pull-to-refresh and infinite-scroll loading for any scrollable widget. It is compatible with Flutter 3.x and Dart 3.

1. **Add Dependency**
    Add `smart_refresher` to your `pubspec.yaml` file:
    ```yaml
    dependencies:
      smart_refresher: ^1.0.0
    ```

2. **Install Package**
    Run the following command in your terminal to fetch the package:
    ```bash
    flutter pub get
    ```

3. **Import the Library**
    Add the import statement to your Dart files where you intend to use the refresher:
    ```dart
    import 'package:smart_refresher/smart_refresher.dart';
    ```
